# Skills Vault

80 проверенных Claude Code skills, разложенных по 9 категориям. Перебрал 307 за месяц — оставил только рабочие. Бесплатно, лицензия MIT.

## Как пользоваться
1. Скачай нужный архив ниже (или `vault-all.zip` — всё сразу).
2. Распакуй в `~/.claude/skills/` (глобально) или в `.claude/skills/` проекта.
3. Перезапусти Claude Code — скиллы подхватятся.
